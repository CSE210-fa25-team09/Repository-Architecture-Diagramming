# Repository Architecture Diagramming

Project 3

## Frontend

## Prerequisites

- Node.js 20.19+, 22.12+. [how to install](https://nodejs.org/en/download)
- npm

## Install

From the repo root:

```bash
cd frontend
npm install
```

## Start

Run the Vite dev server:

```bash
npm run dev
```

Then open the printed local URL (defaults to `http://localhost:5173`).
